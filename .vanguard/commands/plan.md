# Vanguard: Plan Phase

**Project**: vanguard-cli
**Agent**: scrum-master (`.vanguard/agents/scrum-master.md`)

## Planning Phase

Load the Scrum Master Agent persona from `.vanguard/agents/scrum-master.md`.

### Your Task

Break down the technical plan into granular tasks.

1. **Review the plan** at `.vanguard/plans/[feature-name].md`
2. **Create task files** using `.vanguard/templates/task-template.md`
3. **Ensure full context** in each task (no assumptions)
4. **Define dependencies** between tasks
5. **Set acceptance criteria** with checkboxes

### Task Requirements

Each task must:
- Be completable in a single session
- Have clear acceptance criteria
- Include all necessary context
- Reference relevant code examples
- Specify required tests

### Output

Save tasks to `.vanguard/tasks/[feature-name]/task-XXX.md`
