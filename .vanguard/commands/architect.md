# Vanguard: Architect Phase

**Project**: vanguard-cli
**Agent**: architect (`.vanguard/agents/architect.md`)

## Architecture Phase

Load the Architect Agent persona from `.vanguard/agents/architect.md`.

### Your Task

Create a technical plan using the template at `.vanguard/templates/plan-template.md`.

1. **Review the specification** at `.vanguard/specs/[feature-name].md`
2. **Design components** following the architecture pattern
3. **Define data models** and API contracts
4. **Plan implementation phases**
5. **Identify risks and mitigations**

### Architecture Constraints

Reference the constitution at `.vanguard/constitution.md` for:
- Required architecture patterns
- Technology stack constraints
- Quality requirements

### Output

Save the plan to `.vanguard/plans/[feature-name].md`
