# Vanguard: Discover Phase

**Project**: vanguard-cli
**Agent**: analyst (`.vanguard/agents/analyst.md`)

## Discovery Phase

Load the Analyst Agent persona from `.vanguard/agents/analyst.md`.

### Your Task

1. **If brownfield project**: Analyze the existing codebase
   - Document current architecture
   - Identify technical debt
   - Map dependencies
   - Note deviations from intended architecture

2. **If greenfield project**: Explore the problem space
   - Gather requirements through questions
   - Identify constraints and risks
   - Surface hidden assumptions
   - Document success metrics

### Output

Create a discovery document in `.vanguard/specs/discovery.md` with:
- Problem statement
- Constraints identified
- Risks and dependencies
- Open questions
- Recommendations for next steps
