# Vanguard: Specify Phase

**Project**: vanguard-cli
**Agent**: product-manager (`.vanguard/agents/product-manager.md`)

## Specification Phase

Load the Product Manager Agent persona from `.vanguard/agents/product-manager.md`.

### Your Task

Create a specification document using the template at `.vanguard/templates/spec-template.md`.

1. **Understand the feature request** from the user
2. **Define user scenarios** with personas
3. **Document functional requirements** with priorities
4. **Identify non-functional requirements**
5. **Explicitly state what's out of scope**

### Important

- Focus on WHAT and WHY, not HOW
- Each requirement should be testable
- Link requirements to user personas
- Reference the constitution for constraints

### Output

Save the specification to `.vanguard/specs/[feature-name].md`
