# Step 2/4: Identify Tasks

> Break plan into atomic, completable tasks

## Objective

Break plan into atomic, completable tasks

## Inputs Required

- Plan review

## Outputs Produced

- Task list with descriptions

---

Break down each component into tasks:
- Each task should be completable in 1-4 hours
- Tasks should be self-contained
- Include setup tasks (schema, config) first
- Include testing tasks

---

## Checkpoint

Before proceeding to the next step, confirm:

> **Is each task small enough to complete in one session?**

## Tips

- If a task needs more than one paragraph to describe, split it
