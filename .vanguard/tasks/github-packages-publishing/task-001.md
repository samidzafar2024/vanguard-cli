### Required Changes

**package.json modifications**:

```json
{
  "name": "@vanguard-data/vanguard-cli",
  "publishConfig": {
    "registry": "https://npm.pkg.github.com",
    "access": "restricted"
  },
  "bin": {
    "vanguard": "./dist/presentation/cli/index.js",
    "vanguard-cli": "./dist/presentation/cli/index.js"
  },
  "repository": {
    "type": "git",
    "url": "https://github.com/vanguard-cli.git"
  }
}
```

### Key Decisions

1. **`access: restricted`**: Keeps package private (GitHub Packages default for scoped packages)
2. **Dual bin entries**: Allows both `vanguard` and `vanguard-cli` commands for flexibility
3. **Repository URL**: Must match actual GitHub org/repo for GitHub Packages linking
